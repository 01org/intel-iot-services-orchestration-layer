Run iSOL on LattePanda

iSOL runs on the LattePanda Board with Windows 10 inside

Prerequisite

  • LattePanda Board with Windows 10 32bit inside

Configure Windows

  • git-bash: download and install git-bash in https://git-scm.com/downloads
  • Node.js: install Node.js with version >= 0.12.0.
